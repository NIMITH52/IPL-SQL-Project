# IPL Player Analysis using SQL

## Project Overview
This project analyzes IPL player data using SQL Server, focusing on core SQL fundamentals used in Data Analyst roles.

## SQL Concepts Covered
- SELECT
- WHERE
- AND / OR
- IN
- BETWEEN

## Dataset
IPL player dataset containing player name, team, role, price, matches, age, country, and captain status.
